# BigBlind — Poker MTT Trainer 🃏

เทรนเนอร์ฝึกตัดสินใจโป๊กเกอร์ทัวร์นาเมนต์ (ภาษาไทย) — mobile-first, ไม่มี build step

- 30 มือต่อเนื่องเล่นได้ตั้งแต่ preflop ถึง showdown/fold + spot เดี่ยวอีกชุดใหญ่
- ครอบคลุม ICM (bubble / final table / heads-up), bluff, bluff-catch, thin value, cooler
- เฉลย showdown ทุกมือ verify ด้วย `showdown_check.js` (brute-force 990 combos)

เปิดเล่น: เปิด `index.html` ตรงๆ หรือ `python3 -m http.server`
